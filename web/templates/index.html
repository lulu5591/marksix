How to Use

  • History: Browse past draws, search by draw number or date, jump to a page, and click "Update Records" to fetch new draws.
  • Predict: Check a candidate set for duplicates and generate a brand-new non-duplicate combination.
  • Smart Suggest: Use pattern-based generation with recent-trend weighting (3 even, 3 odd; max consecutive run ≤ 2).
  • Insights: View patterns that never occur and rarity stats (parity distribution, consecutive runs, decade and last-digit checks).

All data is stored locally in data/mark_six.sqlite. Re-run updates anytime.
